# transform node 节点

transform node 节点是用来组织场景中的其他节点，或者用来做一些变换操作的对象。

## 公共属性

| 属性名         | 描述   | 类型      | 默认值 |
| -------------- | ------ | --------- | ------ |
| **position**   | 位置   | `Vector3` | -      |
| **rotation**   | 旋转   | `Vector3` | -      |
| **quaternion** | 四元数 | `Vector4` | -      |
| **scale**      | 缩放   | `Vector3` | -      |
| **layer**      | 渲染层 | `Number`  | -      |
